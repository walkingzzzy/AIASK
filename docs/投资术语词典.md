## 基本概念

### 股票代码

A股股票的6位数字代码：

| 开头 | 交易所 | 示例 |
|------|--------|------|
| 600/601/603 | 上海证券交易所 | 600519 贵州茅台 |
| 000/001/002 | 深圳证券交易所 | 000001 平安银行 |
| 300/301 | 创业板 | 300750 宁德时代 |
| 688 | 科创板 | 688981 中芯国际 |

### 涨跌幅

股票当日价格相对于昨日收盘价的变化百分比。

- **涨停**：A股主板涨幅达到10%（ST股票5%）
- **跌停**：A股主板跌幅达到10%（ST股票5%）

### 市值

公司股票的总价值 = 股价 × 总股本

- **大盘股**：市值 > 500亿
- **中盘股**：市值 100-500亿
- **小盘股**：市值 < 100亿

## 财务指标

### PE（市盈率）

**全称**：Price to Earnings Ratio（价格收益比）

**计算**：股价 ÷ 每股收益

**含义**：投资者愿意为每1元利润支付多少钱

**解读**：
| PE范围 | 一般解读 |
|--------|---------|
| < 15 | 估值较低 |
| 15-25 | 估值适中 |
| 25-40 | 估值偏高 |
| > 40 | 估值较高 |

**注意**：不同行业PE标准不同，需同行业对比

### PB（市净率）

**全称**：Price to Book Ratio（价格账面比）

**计算**：股价 ÷ 每股净资产

**含义**：股价相对于公司净资产的倍数

**解读**：
| PB范围 | 一般解读 |
|--------|---------|
| < 1 | 股价低于净资产（可能被低估） |
| 1-3 | 正常范围 |
| > 3 | 估值较高 |

### ROE（净资产收益率）

**全称**：Return on Equity

**计算**：净利润 ÷ 净资产 × 100%

**含义**：公司用股东的钱赚钱的能力

**解读**：
| ROE范围 | 一般解读 |
|---------|---------|
| > 20% | 优秀 |
| 15-20% | 良好 |
| 10-15% | 一般 |
| < 10% | 较差 |

### 毛利率

**计算**：(营业收入 - 营业成本) ÷ 营业收入 × 100%

**含义**：公司产品的盈利空间

**解读**：毛利率越高，说明产品竞争力越强

### 净利率

**计算**：净利润 ÷ 营业收入 × 100%

**含义**：公司最终的盈利能力

### 资产负债率

**计算**：总负债 ÷ 总资产 × 100%

**含义**：公司的负债水平

**解读**：
| 资产负债率 | 一般解读 |
|-----------|---------|
| < 40% | 负债较低，财务稳健 |
| 40-60% | 正常范围 |
| > 70% | 负债较高，需关注风险 |

## 技术指标

### MA（移动平均线）

**全称**：Moving Average

**含义**：一段时间内股价的平均值

**常用周期**：
- **MA5**：5日均线（短期趋势）
- **MA10**：10日均线（短期趋势）
- **MA20**：20日均线（中期趋势）
- **MA60**：60日均线（中长期趋势）

**用法**：
- 股价在均线上方 → 趋势向上
- 股价在均线下方 → 趋势向下
- 短期均线上穿长期均线 → 金叉（买入信号）
- 短期均线下穿长期均线 → 死叉（卖出信号）

### MACD

**全称**：Moving Average Convergence Divergence（指数平滑异同移动平均线）

**组成**：
- **DIF**：快线
- **DEA**：慢线
- **柱状图**：DIF - DEA

**用法**：
- DIF上穿DEA → 金叉（买入信号）
- DIF下穿DEA → 死叉（卖出信号）
- 柱状图由负转正 → 多头趋势
- 柱状图由正转负 → 空头趋势

### RSI（相对强弱指标）

**全称**：Relative Strength Index

**范围**：0-100

**解读**：
| RSI范围 | 含义 | 建议 |
|---------|------|------|
| > 80 | 超买 | 可能回调 |
| 70-80 | 偏强 | 谨慎追高 |
| 30-70 | 正常 | 观望 |
| 20-30 | 偏弱 | 关注机会 |
| < 20 | 超卖 | 可能反弹 |

### KDJ（随机指标）

**组成**：
- **K值**：快速随机值
- **D值**：慢速随机值
- **J值**：方向敏感值

**用法**：
- K上穿D → 金叉（买入信号）
- K下穿D → 死叉（卖出信号）
- J > 100 → 超买
- J < 0 → 超卖

### 布林带（BOLL）

**组成**：
- **上轨**：中轨 + 2倍标准差
- **中轨**：20日均线
- **下轨**：中轨 - 2倍标准差

**用法**：
- 股价触及上轨 → 可能回调
- 股价触及下轨 → 可能反弹
- 带宽收窄 → 可能变盘

### 量比

**计算**：当日成交量 ÷ 过去5日平均成交量

**解读**：
| 量比 | 含义 |
|------|------|
| < 0.5 | 缩量 |
| 0.5-1.5 | 正常 |
| 1.5-3 | 温和放量 |
| 3-5 | 明显放量 |
| > 5 | 巨量 |

## 资金指标

### 北向资金

**含义**：通过沪港通、深港通从香港流入A股的资金

**别名**：外资、聪明钱

**意义**：北向资金流入通常被视为看好A股的信号

### 主力资金

**含义**：大额交易资金，通常指机构投资者的资金

**判断标准**：单笔成交金额 > 50万元

### 资金流向

**计算**：主动买入金额 - 主动卖出金额

**解读**：
- 净流入 > 0 → 资金看好
- 净流出 < 0 → 资金离场

### 融资融券

- **融资**：借钱买股票（看涨）
- **融券**：借股票卖出（看跌）
- **融资余额增加** → 市场看涨情绪
- **融券余额增加** → 市场看跌情绪

## 交易术语

### 集合竞价

**时间**：9:15-9:25

**含义**：开盘前的竞价交易，确定开盘价

**阶段**：
- 9:15-9:20：可挂单可撤单
- 9:20-9:25：可挂单不可撤单
- 9:25：确定开盘价

### 涨停板/跌停板

**含义**：股价达到当日最大涨跌幅限制

**限制**：
- 主板：±10%
- ST股票：±5%
- 科创板/创业板：±20%

### 龙虎榜

**含义**：当日异常交易的股票及其买卖席位信息

**上榜条件**：
- 涨跌幅偏离值 > 7%
- 换手率 > 20%
- 振幅 > 15%

### 换手率

**计算**：成交量 ÷ 流通股本 × 100%

**解读**：
| 换手率 | 含义 |
|--------|------|
| < 1% | 低迷 |
| 1-3% | 正常 |
| 3-7% | 活跃 |
| > 10% | 非常活跃 |

## 信号术语

### AI评分信号

| 信号 | 英文 | 含义 |
|------|------|------|
| 强烈买入 | Strong Buy | 非常看好 |
| 买入 | Buy | 看好 |
| 持有 | Hold | 中性 |
| 卖出 | Sell | 看空 |
| 强烈卖出 | Strong Sell | 非常看空 |

### 夏普比率

**含义**：衡量投资回报与风险的比率

**计算**：(投资收益率 - 无风险收益率) ÷ 收益率标准差

**解读**：
| 夏普比率 | 含义 |
|---------|------|
| > 1 | 优秀 |
| 0.5-1 | 良好 |
| 0-0.5 | 一般 |
| < 0 | 较差 |

### 最大回撤

**含义**：投资过程中从最高点到最低点的最大跌幅

**用途**：衡量投资策略的风险

## 📝 使用提示

1. **不同行业标准不同**：PE、PB等指标需要同行业对比
2. **指标需要综合判断**：单一指标可能有误导性
3. **历史不代表未来**：技术指标基于历史数据
4. **投资有风险**：所有分析仅供参考
