## AI智能评分

### 功能介绍

AI智能评分是系统的核心功能，通过分析股票的多个维度，给出1-10分的综合评分。

### 评分维度

| 维度 | 权重 | 分析内容 |
|------|------|---------|
| 技术面 | 20% | 价格趋势、均线系统、技术形态 |
| 基本面 | 25% | 财务指标、盈利能力、成长性 |
| 资金面 | 20% | 主力资金、北向资金、融资融券 |
| 情绪面 | 15% | 市场情绪、新闻舆情、热度 |
| 风险 | 20% | 波动率、估值风险、流动性 |

### 使用步骤

1. **进入AI评分页面**
   - 点击左侧菜单 "AI评分"

2. **输入股票代码**
   - 在输入框输入6位股票代码
   - 例如：600519（贵州茅台）

3. **开始分析**
   - 点击"开始分析"按钮
   - 等待3-5秒获取结果

4. **查看结果**
   - **综合评分**：1-10分的总体评价
   - **跑赢概率**：预测跑赢大盘的概率
   - **置信度**：评分的可信程度
   - **买卖信号**：Strong Buy / Buy / Hold / Sell / Strong Sell

5. **查看详情**
   - **分项评分**：各维度的具体得分
   - **利好因素**：支撑评分的正面因素
   - **风险因素**：需要注意的风险点
   - **投资建议**：基于分析的操作建议

### 评分解读

| 评分 | 信号 | 含义 | 建议操作 |
|------|------|------|---------|
| 8-10 | Strong Buy | 非常优秀 | 可重点关注，择机买入 |
| 7-8 | Buy | 良好 | 可适当关注 |
| 5-7 | Hold | 一般 | 持有观望 |
| 3-5 | Sell | 较差 | 谨慎持有 |
| 1-3 | Strong Sell | 很差 | 建议回避 |

### 注意事项

- ⚠️ 评分仅供参考，不构成投资建议
- ⚠️ 评分基于历史数据，不代表未来表现
- ⚠️ 建议结合自身判断做出决策

## 技术分析

### 功能介绍

计算并展示股票的各项技术指标，帮助判断买卖时机。

### 主要指标

#### 均线系统（MA）

| 指标 | 周期 | 用途 |
|------|------|------|
| MA5 | 5日 | 短期趋势 |
| MA10 | 10日 | 短期趋势 |
| MA20 | 20日 | 中期趋势 |
| MA60 | 60日 | 中长期趋势 |

**判断方法**：
- 股价在均线上方 → 趋势向上
- 股价在均线下方 → 趋势向下
- 短期均线上穿长期均线 → 金叉（买入信号）
- 短期均线下穿长期均线 → 死叉（卖出信号）

#### MACD指标

| 组成 | 说明 |
|------|------|
| DIF | 快线，反映短期趋势 |
| DEA | 慢线，反映中期趋势 |
| 柱状图 | DIF与DEA的差值 |

**判断方法**：
- DIF上穿DEA → 金叉（买入信号）
- DIF下穿DEA → 死叉（卖出信号）
- 柱状图由负转正 → 多头趋势开始
- 柱状图由正转负 → 空头趋势开始

#### RSI指标

| 区间 | 状态 | 建议 |
|------|------|------|
| > 80 | 超买 | 可能回调，谨慎追高 |
| 70-80 | 偏强 | 关注回调风险 |
| 30-70 | 正常 | 正常交易 |
| 20-30 | 偏弱 | 关注反弹机会 |
| < 20 | 超卖 | 可能反弹 |

#### KDJ指标

| 信号 | 条件 | 含义 |
|------|------|------|
| 金叉 | K上穿D | 买入信号 |
| 死叉 | K下穿D | 卖出信号 |
| 超买 | J > 100 | 可能回调 |
| 超卖 | J < 0 | 可能反弹 |

### 使用步骤

1. 进入"技术分析"页面
2. 输入股票代码
3. 点击"计算指标"
4. 查看各项指标数值和判断

## 智能问答

### 功能介绍

用自然语言提问，AI帮您分析股票、筛选股票、查询数据。

### 支持的问题类型

| 类型 | 示例问题 |
|------|---------|
| 个股分析 | "分析贵州茅台" |
| 数据查询 | "茅台的PE是多少" |
| 条件筛选 | "找出PE低于20的股票" |
| 推荐股票 | "推荐几只消费股" |
| 对比分析 | "比较茅台和五粮液" |

### 使用技巧

1. **问题要具体**
   - ✅ "分析600519的基本面"
   - ❌ "这只股票怎么样"

2. **可以用股票代码或名称**
   - "分析600519" 和 "分析贵州茅台" 都可以

3. **筛选条件要明确**
   - ✅ "找出PE低于20且ROE大于15%的股票"
   - ❌ "找出好股票"

### 示例对话

```
用户：分析贵州茅台

AI：正在分析贵州茅台(600519)...

【基本信息】
- 当前价格：1800.00元
- 市值：22600亿元
- 所属行业：白酒

【AI评分】8.5/10
- 技术面：8.0分
- 基本面：9.0分
- 资金面：8.5分

【投资建议】
贵州茅台基本面优秀，ROE持续保持在30%以上...
```

## 选股雷达

### 功能介绍

根据预设策略或自定义条件，从全市场筛选符合条件的股票。

### 预设策略

| 策略名称 | 筛选条件 | 适合人群 |
|---------|---------|---------|
| AI高分股 | AI评分 ≥ 7分 | 追求综合质量 |
| 低估值股 | PE < 15, PB < 2 | 价值投资者 |
| 高成长股 | 营收增长 > 20% | 成长投资者 |
| 高股息股 | 股息率 > 3% | 稳健投资者 |
| 强势股 | 近期涨幅领先 | 趋势投资者 |

### 使用步骤

1. 进入"选股雷达"页面
2. 选择预设策略或自定义条件
3. 设置返回结果数量
4. 点击"开始筛选"
5. 查看筛选结果列表
6. 点击个股查看详情

## 组合管理

### 功能介绍

管理您的股票持仓，跟踪收益，分析风险。

### 主要功能

#### 持仓概览

显示您的投资组合整体情况：
- **总市值**：所有持仓的当前市值
- **总成本**：所有持仓的买入成本
- **总盈亏**：盈利或亏损金额
- **盈亏比例**：收益率百分比

#### 添加持仓

1. 点击"添加持仓"标签
2. 输入股票代码和名称
3. 输入持仓数量和成本价
4. 点击"添加持仓"按钮

#### 删除持仓

1. 在持仓列表中选择要删除的股票
2. 点击"删除"按钮

#### 风险分析

系统会分析您的持仓风险：
- **最大单仓占比**：单只股票占比是否过高
- **持仓数量**：是否过于集中或分散
- **风险等级**：低/中/高
- **风险建议**：具体的优化建议

### 风险等级说明

| 风险等级 | 条件 | 建议 |
|---------|------|------|
| 低 | 单仓 < 30%，持仓 ≥ 5只 | 分散良好 |
| 中 | 单仓 30-50%，持仓 3-5只 | 适度分散 |
| 高 | 单仓 > 50%，持仓 < 3只 | 建议分散 |

## 回测分析

### 功能介绍

验证投资策略在历史数据上的表现。

### 主要指标

| 指标 | 说明 | 好的标准 |
|------|------|---------|
| 总收益率 | 策略的总体收益 | > 基准收益 |
| 年化收益 | 换算成年度收益 | > 10% |
| 最大回撤 | 最大亏损幅度 | < 20% |
| 夏普比率 | 风险调整后收益 | > 1 |
| 胜率 | 盈利交易占比 | > 50% |

### 使用步骤

1. 进入"回测分析"页面
2. 选择回测策略
3. 设置回测时间范围
4. 设置初始资金
5. 点击"开始回测"
6. 查看回测结果和收益曲线

### 注意事项

- ⚠️ 历史表现不代表未来收益
- ⚠️ 回测结果可能存在过拟合
- ⚠️ 实际交易还需考虑交易成本

## 涨停追踪

### 功能介绍

追踪当日涨停股票，分析涨停原因和连板情况。

### 主要信息

- **涨停股票列表**：当日涨停的所有股票
- **涨停原因**：分析涨停的主要原因
- **连板天数**：连续涨停的天数
- **封单金额**：涨停封单的资金量
- **开板次数**：涨停期间打开的次数

### 使用场景

- 发现市场热点
- 追踪强势股票
- 分析板块轮动

## 资金监控

### 功能介绍

监控市场资金流向，包括主力资金、北向资金等。

### 主要数据

| 数据类型 | 说明 |
|---------|------|
| 主力资金 | 大单资金的买卖情况 |
| 北向资金 | 外资通过沪深港通的流入流出 |
| 融资融券 | 两融资金的变化 |
| 板块资金 | 各行业板块的资金流向 |

### 资金信号解读

| 信号 | 含义 |
|------|------|
| 主力净流入 | 大资金看好，可能上涨 |
| 主力净流出 | 大资金离场，可能下跌 |
| 北向净买入 | 外资看好A股 |
| 北向净卖出 | 外资减仓A股 |

## 竞价分析

### 功能介绍

分析集合竞价阶段的交易数据，发现早盘机会。

### 竞价时间

- **9:15-9:20**：可挂单可撤单
- **9:20-9:25**：可挂单不可撤单
- **9:25**：确定开盘价

### 主要指标

| 指标 | 说明 |
|------|------|
| 竞价涨幅 | 竞价价格相对昨收的涨跌幅 |
| 竞价量比 | 竞价成交量与历史对比 |
| 买卖单比 | 买单和卖单的比例 |
| 大单占比 | 大额订单的占比 |

### 使用场景

- 发现早盘强势股
- 判断开盘走势
- 寻找竞价异动股
