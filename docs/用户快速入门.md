## 安装方式

### 方式一：在线使用（推荐新手）

如果您不想安装软件，可以直接访问在线版本：
- 网址：[待部署]
- 无需安装，打开浏览器即可使用

### 方式二：本地安装

如果您希望在自己电脑上运行，请按以下步骤操作：

#### 第一步：安装Python

1. 访问 [Python官网](https://www.python.org/downloads/)
2. 下载 Python 3.12 或更高版本
3. 安装时勾选 "Add Python to PATH"

#### 第二步：安装uv包管理器

打开命令提示符（Windows）或终端（Mac），输入：

```bash
# Windows
pip install uv

# Mac/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
```

#### 第三步：下载并运行

```bash
# 1. 下载项目
git clone [项目地址]
cd easy_investment_Agent_crewai

# 2. 安装依赖
uv sync

# 3. 配置环境
cp .env.example .env
# 编辑 .env 文件，填入您的API密钥

# 4. 启动系统
uv run stock-app
```

#### 第四步：打开浏览器

启动成功后，打开浏览器访问：`http://localhost:8501`

### 2️⃣ 尝试第一次分析

1. 点击左侧菜单的 **"AI评分"**
2. 在输入框输入股票代码，如 `600519`（贵州茅台）
3. 点击 **"开始分析"** 按钮
4. 等待几秒，查看分析结果

### 3️⃣ 理解AI评分

AI评分是1-10分的综合评估：

| 评分范围 | 含义 | 建议 |
|---------|------|------|
| 8-10分 | 优秀 | 可重点关注 |
| 6-7分 | 良好 | 可适当关注 |
| 4-5分 | 一般 | 谨慎对待 |
| 1-3分 | 较差 | 建议回避 |

评分由5个维度组成：
- **技术面**：价格走势、技术指标
- **基本面**：财务状况、盈利能力
- **资金面**：主力资金、北向资金流向
- **情绪面**：市场情绪、新闻舆情
- **风险**：波动性、估值风险

## 核心功能速览

### 🎯 AI智能评分

**用途**：快速了解一只股票的综合质量

**操作步骤**：
1. 输入股票代码（如 600519）
2. 点击"开始分析"
3. 查看综合评分和分项评分
4. 阅读利好因素和风险提示

**小贴士**：
- 评分仅供参考，不构成投资建议
- 建议结合多个维度综合判断

### 📊 技术分析

**用途**：查看股票的技术指标

**主要指标说明**：
- **MA（均线）**：价格的平均趋势
- **MACD**：判断买卖时机的指标
- **RSI**：判断超买超卖的指标
- **KDJ**：短期买卖信号指标

### 💬 智能问答

**用途**：用自然语言提问

**示例问题**：
- "分析贵州茅台"
- "找出PE低于20的股票"
- "茅台的PE是多少"
- "推荐几只消费股"

### 🔍 选股雷达

**用途**：根据条件筛选股票

**预设策略**：
- AI高分股：评分7分以上的股票
- 低估值股：PE、PB较低的股票
- 高成长股：营收增长较快的股票

### 💼 组合管理

**用途**：管理您的持仓

**功能**：
- 添加/删除持仓
- 查看盈亏情况
- 风险分析

## 常见问题

### Q: 股票代码怎么输入？

A: 输入6位数字即可，例如：
- 上交所：600519（贵州茅台）
- 深交所：000001（平安银行）
- 创业板：300750（宁德时代）

### Q: 分析结果准确吗？

A: AI评分基于公开数据计算，仅供参考。投资决策请结合自身判断，本系统不构成投资建议。

### Q: 数据多久更新一次？

A: 行情数据实时更新（交易时间内），财务数据按季度更新。

### Q: 遇到错误怎么办？

A: 常见解决方法：
1. 检查网络连接
2. 确认股票代码正确
3. 稍后重试
4. 如问题持续，请联系技术支持
