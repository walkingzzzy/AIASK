# akshare-fund-news 环境变量

来自项目 README 与工具实现：
- TUSHARE_TOKEN: Tushare Pro token（北向资金优先使用）
- NORTH_FUND_STALE_DAYS: 北向资金最大允许滞后天数（默认 5）
- NORTH_FUND_DAILY_QUOTA: 北向资金日额度（默认 52000000000）
- HKEX_DAILY_STAT_URL: 港交所日度数据 URL 模板
- AKSHARE_NOTICE_MAX_DAYS: 公告最大日期跨度（默认 31）
- AKSHARE_NOTICE_MAX_SECONDS: 公告抓取最大耗时秒数（默认 20）
- AKSHARE_NOTICE_RETRY: 公告抓取重试次数（默认 2）
